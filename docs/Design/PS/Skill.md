# PS技巧收录

## 测量

测量距离：使用标尺工具，然后按住shift可以保持直线，在顶部信息栏中就能看到具体长度。

## 图层

### 选择图层

## 其他

拖动画布： 按住空格+拖动鼠标

拖动画板： 按住空格,然后拖动

缩放： alt+滚轮

重命名组：在组名上双击即可。
